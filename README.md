# OPSBOX Core System

AI-powered transportation logistics system.

## System Components

### Agents
- Dispatch Agent: Order management and assignment
- Routing Agent: Route optimization and navigation
- Fleet Agent: Vehicle and resource management
- Supervisor Agent: System oversight and coordination

### Directory Structure
```
opsbox-core/
├── agents/
│   ├── dispatch/
│   ├── routing/
│   ├── fleet/
│   └── supervisor/
├── services/
├── data/
└── config/
```

## Version
Current version: 1.0.0

## Setup
1. Initialize the system
2. Configure agents
3. Start services
4. Deploy agents

## Configuration
See `config/system.json` for system-wide settings.
